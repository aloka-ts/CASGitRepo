Starting Embedded CAS

This option explains you how to start the embedded CAS.  User can start Embedded CAS either in RUN mode or in DEBUG mode. By starting embedded CAS in debug mode user can do the debugging.(Java Debug Perspective is opened. For using Java Debug Perspective refer to Eclipse debugging )

To start embedded a CAS (RUN mode)

1.    Open Eclipse.

2.    Click on CAS Management menu > Select Embedded CAS > Start CAS. User can start in Debug mode by clicking CAS Management menu > Select Embedded CAS > Start CAS (Debug mode)
