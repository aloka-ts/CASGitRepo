Deploying a Service

This option allows you to deploy a service. Once the service is build the user can deploy that service.  To deploy the service the user needs to first build the project, build the service and then deploy it.

To deploy a service

Open Eclipse.

Click on Project menu > Build Project.

Click on Build Service menu > Build Service.

Eclipse will display the message box, as shown in the following figure.

     

Click the OK button.

Click on Build Service menu > Deploy Service.

Eclipse will display the Bay Packets CAS Services Deployment window, as shown in the following figure. if user wants to deploy the service on a different running instance of CAS (non Embedded) then select the Different Running Instance of CAS. Enter the <IPAddress>:<JMX PORT> in the host address text box.

     

Note: You can also deploy the service by clicking Deploy Service button in Service Management tab.     

Enter the appropriate information and click the Deploy button.
