Creating a Project

This option allows you to create a Agnity project. 

To create a project

1.    Open Eclipse.

2.    Click on File menu > Select New > Project.

3.    Eclipse will display New Project window, as shown in the following figure.

    

4.    Select Agnity Project, under Agnity IDE and click the Next button.

5.    Eclipse will display Create Agnity Project window, as shown in the following figure.

 

 

6.    Type the appropriate project name in Project name field and click the Next button.

7.    Eclipse will display Specify Application Type window, as shown in the following figure.

 

 

Fields

Description

Application Info

Name

Type the name of the project.

By default, this field displays the project name.

Version

Type the version number.

By default this field displays the version number.

Priority

Select the priority from the drop-down list.

Application Type

SIP Application

Select/clear the check box.

If its selected, then Agnity will provide SIP based application.

By default, this check box is selected.

HTTP Application

Select/clear the check box.

If its selected, then Agnity will provide HTTP based application. (JBOSS plugin, bundled with this plugin is used for the development of HTTP based services)

By default, this check box is selected.

Use Agnity Service Building Blocks

Select/clear the check box.

If its selected, then Agnity will provide Service Building Blocks (SBB) for SIP based application development.

By default, this check box is selected.

 

8.    Enter the appropriate details and click the Next button.

9.    Eclipse will display Java Settings - Source window, as shown in the following figure.

 

 

10.                        To add more folders in the Source tab, click the Add Folder… button.

11.                        Click the Projects tab.

12.                        Eclipse will display Java Settings - Project window, as shown in the following figure.

 

 

Fields

Description

Add JARs…

Click this button to add the existing project.

Edit

Click this button to edit the existing project under Project tab.

Remove

Click this button to remove the existing project under Project tab.

Browse

Click this button to browse the desired location for the output folder.

 

13.                        Enter the appropriate details and click the Libraries tab.

14.                        Eclipse will display Java Settings - Libraries window.

 

 

Fields

Description

Add JARs…

Click this button to add the JAR files from the existing project.

Add External JARs…

Click this button to add the external JAR files.

Add Variable…

Click this button to add the variables.

Add Library…

Click this button to add the library.

Add Class Folder…

Click this button to add the class folder.

Edit

Click this button to edit the existing files under libraries tab.

Remove

Click this button to remove the existing files under libraries tab.

Browse

Click this button to browse the desired location for the output folder.

 

15.                       Enter the appropriate information and click the Order and Export tab.

16.                       Eclipse will display Java Settings – Order and Export window, as shown in the following figure.

 

 

Fields

Description

Up

Click this button to move the files up as per the most priority.

Down

Click this button to move the files down as per the least priority.

Select All

Click this button to select all the files.

Deselect All

Click this button to de-select all the files.

Add Class Folder…

Click this button to add the class folder.

Browse

Click this button to browse the desired location for the output folder.

 

17.                       Enter the appropriate details and click the Finish button.
