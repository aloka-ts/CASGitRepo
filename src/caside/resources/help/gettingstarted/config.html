Configuration Settings

Before using BayPackets SIP Application Server IDE, the user must change the configuration settings. This option allows you to set the configuration parameters.

Important: The user must set the LD_LIBRARY_PATH to the $ASE_HOME/lib/lib before starting the Eclipse.

To set the configuration parameters

1.  Open Eclipse.

2.  Click on Window > Select Preferences.

3.  Eclipse displays the Preferences window, as shown in the following figure.

  

4.  In the left panel, Click Java > Click Build Path > Select Classpath Variables.

5.  Eclipse displays the Preferences window, as shown in the following figure.

   

6.  Click the New button.

7.  Eclipse displays the New Variable Entry window, as shown in the following figure.

8.  Enter the Name as ASE_HOME and the Path should be Agility SIP Application Server Install root.

9.  Click the OK button.
