Starting a Service

This option allows you to start a service. The user needs to first create a service, deploy it and then start that service.

To start a service

1.    Open Eclipse.

2.    Click the Service Management tab.

3.    Eclipse displays the Service Management tab, as shown in the following figure.

    

Note: You can start the service only if the service is in Installed status, as shown in the above figure.           

4.    Click the Start Service button.

5.    The status of the service will change to Ready status, as shown in the following figure.
