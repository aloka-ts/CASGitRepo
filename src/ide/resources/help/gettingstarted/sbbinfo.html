The Service Building Blocks are the BayPackets’ extensions provided on top of the SIP Servlet APIs. The service building blocks are the reusable software components that provide the default handling of the SIP messages so that the application writer can focus on implementing the business logic. They also allow the application developer to extend the functionalities provided by them using the standard SIP servlet APIs. 

IDE provides a wizard for generating template of SBB Event Listener.

 

Some of the examples for SBB are:

  • B2BUA Session Controller SBB
  • Media Server Session Controller SBB
  • Conference Controller SBB
  • Conference Participant SBB
  • TBCT SBB
  • Call Transfer SBB
